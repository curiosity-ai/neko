# Stats Simple

The `stats-simple` component displays simple key-value statistics.

## Usage

```markdown
[!stats-simple
    "Total Subscribers" "71,897"
    "Avg. Open Rate" "58.16%"
    "Avg. Click Rate" "24.57%"
]
```
