# Header Stats

The `header-stats` component displays a header section with key statistics.

## Usage

```markdown
[!header-stats
    title="Trusted by creators worldwide"
    subtitle="We help creators build their business and reach more people."
    "Creators on the platform" "8,000+"
    "Flat platform fee" "3%"
    "Uptime guarantee" "99.9%"
    "Paid out to creators" "$70M"
]
```

## Attributes

| Attribute | Description |
| :--- | :--- |
| `title` | The main title. |
| `subtitle` | The subtitle text. |
| Positional Arguments | Groups of 2: Label, Value. |
