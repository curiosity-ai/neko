# Add components

Neko ships with [a rich set of components](/components/components). Drop them into any Markdown file:

```markdown
!!! info
This is an info callout.
!!!

::: card {title="Hello" icon="rocket"}
A card with an icon badge.
:::
```

Browse the component reference to discover hero blocks, lessons, force graphs, mermaid diagrams and more.
