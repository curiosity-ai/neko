## About this track

This track is a sample of the `[!lesson]` Markdown component. Each step in the curriculum above is a sibling Markdown file inside this folder &mdash; Neko discovers them automatically and reads their `title`, `order`, `kind` and `duration` from the frontmatter.

Your progress is saved in your browser's `localStorage`, so checkmarks stick across reloads. Use **Continue** to jump to the next unfinished step, or **Start over** to reset.
