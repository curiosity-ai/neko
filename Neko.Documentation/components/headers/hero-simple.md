# Hero Simple

The `hero-simple` component displays a simple hero section with title, subtitle, and buttons.

## Usage

```markdown
[!hero-simple
    title="Data to enrich your online business"
    subtitle="Anim aute id magna aliqua ad ad non deserunt sunt. Qui irure qui lorem cupidatat commodo. Elit sunt amet fugiat veniam occaecat fugiat aliqua."
    cta1-text="Get started"
    cta2-text="Learn more"
]
```

## Attributes

| Attribute | Description | Default |
| :--- | :--- | :--- |
| `title` | Main title text. | |
| `subtitle` | Subtitle text. | |
| `cta1-text` | Primary button text. | |
| `cta1-link` | Primary button link. | `#` |
| `cta2-text` | Secondary button text. | |
| `cta2-link` | Secondary button link. | `#` |
