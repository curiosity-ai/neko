# Bento Grid

The `bento-grid` component displays items in a bento-style grid.

## Usage

```markdown
[!bento-grid
    title="Mobile friendly"
    subtitle="Bento grid example."
    "Mobile friendly" "Lorem ipsum dolor sit amet, consectetur adipiscing elit. In gravida justo et nulla efficitur, a aliquet velit." "https://cdn2.thecatapi.com/images/20f.png"
    "Performance" "Lorem ipsum dolor sit amet, consectetur adipiscing elit." "https://cdn2.thecatapi.com/images/db3.jpg"
    "Security" "Lorem ipsum dolor sit amet, consectetur adipiscing elit." "https://cdn2.thecatapi.com/images/e12.jpg"
    "Integrations" "Lorem ipsum dolor sit amet, consectetur adipiscing elit." "https://cdn2.thecatapi.com/images/d45.jpg"
]
```

## Attributes

| Attribute | Description |
| :--- | :--- |
| `title` | The main title. |
| `subtitle` | The subtitle or intro text. |
| Positional Arguments | Groups of 3: Title, Description, Image URL. |
