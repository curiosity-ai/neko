# Icon

Similar to the [Emoji](emoji.md) `:shortcode:` syntax, you can add icons using `:icon-shortcode:`, where the `shortcode` is a [UIcon](uicons.md) icon name.

For example, to add the :icon-star: `star` icon, use the `:icon-star:` icon shortcode.

Icons can be added anywhere within your document, including paragraph text, headings, table content, [Panel](panel.md) component titles, [Callouts](callout.md), [code block](code-block.md) titles, etc. :icon-thumbs-up:

Some icon shortcode names are made up of `two-or-more` words. For instance, the :icon-rocket: `rocket` icon would be configured as `:icon-rocket:`.
