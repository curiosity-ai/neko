# Embed

The Embed component allows you to insert external content like interactive demos, maps, or other iframes.

## Syntax

Use the `[!embed ...]` syntax.

```markdown
[!embed src="https://example.com" height="500"]
```

## Attributes

| Attribute | Description | Default |
| :--- | :--- | :--- |
| `src` | The URL of the content to embed. | (Required) |
| `height` | The height of the embed in pixels. | `400` |

### Example

[!embed src="https://example.com" height="400"]
