# YouTube

Embed YouTube videos easily using the `[!youtube ...]` component.

## Syntax

Use the `[!youtube id="..."]` syntax, where `id` is the video ID from the YouTube URL (e.g., `v=VIDEO_ID`).

```markdown
[!youtube id="VIDEO_ID"]
```

## Attributes

| Attribute | Description | Default |
| :--- | :--- | :--- |
| `id` | The YouTube video ID. | (Required) |
