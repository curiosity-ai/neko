# File download

`file-download` is the same component as [`file`](../file/SKILL.md). The
inline shortcode is `[!file …]` regardless of which name you use.

## Quick reminder

```markdown
[!file](/assets/data.csv)
[!file "Annual report"](/assets/report.pdf)
[!file icon="file-pdf" text="Spec" size="450 KB"](/assets/spec.pdf)
```

See the [`file`](../file/SKILL.md) skill for the full attribute list and tips.
