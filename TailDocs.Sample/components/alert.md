# Alert

Alerts (or Callouts) are used to highlight important information.

## Syntax

You can use the `!!!` syntax or the `> [!NOTE]` syntax.

### Default (Info)

!!! info
This is an information alert.
!!!

### Warning

!!! warning
This is a warning alert.
!!!

### Danger

!!! danger
This is a danger alert.
!!!

### Success

!!! success
This is a success alert.
!!!
