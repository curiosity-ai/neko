# Button

Buttons are used for actions or prominent links.

## Syntax

Use `[!button text="Click Me" link="/"]`.

### Examples

- Default: [!button text="Primary" link="#"]
- Secondary: [!button text="Secondary" variant="secondary"]
- Outline: [!button text="Outline" variant="outline"]
- Danger: [!button text="Danger" variant="danger"]

### With Icon

[!button text="Search" icon="search" variant="outline"]
