# Icon

The Icon component allows you to insert Flaticon UIcons inline.

## Syntax

`[!icon name="home"]` or inline `:icon-home:` if supported.

### Examples

- Home: [!icon name="home"]
- User: [!icon name="user"]
- Settings: [!icon name="settings"]

You can use any valid UIcons name available in the linked CSS.
